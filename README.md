# LetMeIn
This is a web application that focus on user Authentication using username and password by applying different methods, such as, Encryption, Hashing, Salting and Hashing, Passport.js, and OAuth 2.0.

# Tools Used
- HTML
- CSS
- Javascript
- Bootstrap
- Node.js
- Express.js
- EJS
- Mongodb + Mongoose
